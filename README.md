# Feel depressed?

Let yourself get sad by looking at the bubbles and popping them to calm, un-copyrighted Lo-Fi (and not only Lo-Fi) music.

### Hotkeys

**\[F1\]** - Show/Hide UI

**\[SPACE\]** - Play/Pause music

**\[Ctrl+N\]** - Next track

**\[Ctrl+S\]** - Enable/Disable tracks shuffling
